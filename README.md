## Desenvolvimento

`./gradlew yarn_install`

Instala o `yarn` e dependências no cache do Gradle.

`./gradlew yarn_build`

Constrói os artefatos em `/public`.

`./gradlew yarn_run_dev`

Constrói os artefatos e serve de um servidor local para desenvolvimento.
